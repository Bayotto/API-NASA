# Nasa APOD Viewer

Uma aplicação web construida REACT e Vite que consome a API Astronomy Picture of they Day (APOD) da NASA. O projeto exibe diariamente uma imagem (ou vídeo) do universo, acompanhando de título, descrição e outras informações.
